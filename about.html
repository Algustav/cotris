Cotris

与 Codex 协力完成的俄罗斯方块游戏。

返回游戏 主题预览 Todo
  • Project COTRIS 是一个人工 0 代码的个人项目，也是我的第一个完整 vibecoding 作品，总投入 4 天，每天 2 小时；主框架半小时搭好，剩下时间几乎都在功能调优、美化和抓 bug。
  • Name CO 来自 coding + cooperation，再加上 teTRIS。它是我与 Codex 协作的第一个个人游戏作品。
  • Modes 游戏包含 3 种模式：经典马拉松、不会结束的放松禅境，以及方块不会下坠的练习模式。
  • Themes 当前有 10 个模板，7 个深色、3 个浅色，灵感来自 IDE 的经典配色。
  • Device 移动端能玩，但更推荐 PC 或平板。
  • Effects 游戏里做了一些不错的视觉特效，让移动、落地、消行和切换主题更有反馈。
  • Audio 效果音来自 Apotris，背景音乐来自 Suno 生成曲。
  • Release 当前 release 1.6，暂不支持 T-spin / T 旋，因为自己还没玩明白 :)
